Épreuve E6 — Parcours Professionnel

Présentation

L’épreuve E6 « Parcours de professionnalisation » évalue les compétences acquises au cours des missions réalisées en entreprise. Chaque mission est illustrée par un dossier comprenant la documentation technique, les tests et les fiches de réalisation.

Contenu attendu (détaillé)

  • Deux missions professionnelles documentées avec contexte, objectifs et contraintes.
  • Documentation technique et utilisateur (installation, architecture, schémas, captures).
  • Fiche descriptive et fiche de réalisation (rôle, tâches effectuées, durée, outils).
  • Analyse personnelle du parcours, bilan des compétences et axes d'amélioration.
  • Preuves : extraits de code, captures d'écran, logs, rapports de test, livrables (PDF/ZIP).

Mes Projets (E6)

Ci-dessous les deux projets retenus pour l'épreuve.

RMS Ticket

Entreprise / Période — Rôle : Développeur

Système de gestion de tickets pour le support interne : création, suivi, affectation, priorisation et historique des interventions.

  • Responsabilités : Conception des modèles, développement backend, création de l'interface de suivi, tests fonctionnels.
  • Technologies : PHP, JavaScript, HTML/CSS, MySQL (exemples).
  • Base de données : Modèle relationnel avec tables utilisateurs, tickets, commentaires, historiques.
  • Preuves : Captures d'écran, extraits de code, fiche de réalisation (à joindre).
  • Lien : À venir — dépôt GitHub / URL du serveur

Le Muscle Sympa

Entreprise / Période — Rôle : Développeur / Intégrateur

Application ou site pour la gestion d'un club sportif : planning des séances, suivi des adhérents, gestion des abonnements.

  • Responsabilités : Intégration UI, connexion à la base, fonctions d'administration, documentation utilisateur.
  • Technologies : HTML/CSS, JavaScript, possible backend (ex : Node/PHP).
  • Base de données : Tables adhérents, séances, paiements, historiques.
  • Preuves : Captures, rapports de tests, fiche de réalisation (à joindre).
  • Site en ligne : http://lms.testoxylog.fr/
  • Code source : GitHub - Le Muscle Sympa
